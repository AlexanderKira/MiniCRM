Этот HTML файл отображает данные полученные из UserController в виде таблицы 

с учетом маршрутов на кнопки создания, редактирования и удаления создаётся ссылка
пример: href='index.php?page=users&action=create' 

здесь реализованны шаблоны 
шаблон из layout.php в котором реализваны шабка и подвал интерфейса 
в него можно передавать шаблон title с информацией о каждой странице 

таблица состаит из заголовков и колонок

по мимо таблицы здесь реализваны кнопки создания, редактирования и удаления пользователя 
и цикл для работы с разбором массива users на колонки с данными user из модели UserModels и его метода readAll
