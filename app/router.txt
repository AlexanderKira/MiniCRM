отвечает за маршруты проекта

класс который будет запускать функцию run 

run будет спрашивать какой метод будет реализован в адрестной строке 
проверяет если нет дополнительных методов контроллера HomeController, то вернется страница 'home'
если 'users', то уже методы по его контроллеру UserController.

если условия методов не будут соблюдены, выйдет ошибка о не существующей странице
