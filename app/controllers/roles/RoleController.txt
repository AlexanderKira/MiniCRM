контроллер задач свяных с ролями пользователя

подключаем модель Role.php

создаём класс RoleController

метод index использует метод getAllRoles класса Role 
    и подключает к HTML файлу, который будет показывать все созданные роли пользователей

метод create переводит на HTML форму создания ролей

метод store принимает данные с формы create, 
    и с помощью метода createRole класса модели Role создаёт данные в таблице roles
        возвращает на страницу создания ролей

метод Edit с помощью экземпляра getRoleById класса Role находит по id соответсвующую роль
    и выводит их в форму редактирования 

метод update принимает данные с формы и 
    с помощью экземпляра updateRole класса Role редактирует данные в таблице roles
        возвращает на сраницу roles/index.php

метод Delete принмает id и с помощью экземпляра deleteRole класса Role удаляет роль из таблицы roles
    возвращает на сраницу roles/index.php
