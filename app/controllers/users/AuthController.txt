этот файл будет отвечать за авторизацию пользователя

создаём и подключаем модель AuthUser
в классе AuthController создаём метод register который будет показывать HTML форму регистрации

создём метод store который принимает значения из HTML формы username, email, password, confirm_password.
возвращает на страницу login 

создаём метод login, он возвращает HTML страницу авторизации

создаём метод authenticate, когда зарегестрированный пользователь авторизуется у него записываются куки 
в этом методе мы обращаемся к классу модели AuthUser
Прописываем прверку, если данные email и password совпадают с данными из поиска в БД. 
Если данные совпадают, то прописываем этом пользователю его возможности.
проверка заключается по поиску email через модель AuthUser и её метод findByEmail
findByEmail находит и проверяет правильное заполнение пароля 
если пороль совпадает, то делаем session_start
заисываем данные сессии в user_id и user_role в глобальный массив $_SESSION 
от роли будет известно, что доступно пользователю
также есть переменная $remember отвечающая за 'запомни меня' 
если $register равенн 'on', то будут устанавливаться куки 
далее делаем переадресацию например на главную страницу index.php

содаём метод logout, который запускает и уничтожает сессию пользователя
делает переадресацию на index.php
