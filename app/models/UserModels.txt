Это модель запроса к базе данных для регистрации пользователя

создаем класс 
в нем приватную перменную db

создаем метод __construct
в котором мы присваиваем данные метода базы данных и его соединение в переменную

создаем публичный метод readALL который будет обращаться к переменной db
о составленному запросу вызывать изменять или удалять данные 
переменная result будет вызывать данные из таблицы users 
массив result с помощью цикла мы разбиваем на строки в зависимости от длины массива 
с помощью глобальной функции fetch_assoc заисываем все в переменую массива users и возвращаем её в этой функции.

создаём публичный метод create который получает данные с HTML формы с помощью глобального метода пост и переменной data
получаем данные по названию данных формы 
login 
тут же хэшируем password 
данные поля is_admin делаем проверку на 0 или 1. Является ли пользователь админом.
и создаем переменную с глобальной функцией времени

создаем переменую с запросом в переменую базы данных для записи данных из HTML формы 
передаем условные ключи, перечесляем все перменные 

отправляем данные с проверкой. 
если есть данные, то отправляем true иначе false
