Регистрация пользователя

Середина между базой данных и выводом пользователю

подключаем файл модели запросов к бд регистрации пользователя

создаем класс UserConroller

создаём метод index который будет вызывать всех пользователей
Создаём экземпляр класса модели UserModels 
вызываем метод readAll из класса модели UserModels, который будет вытягивать данные из БД и определяем его в перерменую users
в этом же метотоде подключаем HTML файл index.php, в котором будут отображаться данные users

создаём метод create к которому подключаем HTML форму создания пользователя

создаём метод store который будет отправлять данные с формы
создаем в нем проверку на заполнение полей с данными  
а также проверку совпадают ли поля password и confirm_password 
создаем экземпляр класса UserModels и его метод create с глобальным массивом $_POST
